Authentication
/api/auth/user
GET
User Authentication
Request
			Headers: Authentication Header with Token
Response
			Media type: application/json
			Type: object

			Properties
				user: {
					id: number
					username: string
					email: string
				}
/api/auth/login
POST
Log In User
Request
			Headers: Content-Type: application/json
			Properties
				username: string
				password: string
Response
			Media type: application/json
			Type: object

			Properties
				user: {
					id: number
					username: string
					email: string
				}
				token: string
/api/auth/logout
POST
End Of User Session
Request
			Headers: Authentication Header with Token
Response
			204 No Content

POST

			Headers: Content-Type: application/json
			Properties

				email: string

			Media type: application/json
			Type: object
